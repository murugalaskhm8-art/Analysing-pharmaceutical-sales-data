# Analysing Pharmaceutical Sales Data

## Project Overview

This project analyzes pharmaceutical sales data using Python, Pandas, and Matplotlib.

## Dataset

Pharmaceutical Sales Dataset from Kaggle:
https://www.kaggle.com/datasets/milanzdravkovic/pharmaceutical-sales-data

## Technologies Used

* Python 3
* Pandas
* Matplotlib
* Jupyter Notebook

## Analysis Performed

1. Total sales quantities for each drug category (ATC code)
2. Highest-selling drug categories
3. Top 3 drugs in January 2015
4. Top 3 drugs in July 2016
5. Top 3 drugs in September 2017
6. Most sold drug in 2017
7. Highest average daily sales category
8. Monthly analysis of respiratory drugs (R03)
